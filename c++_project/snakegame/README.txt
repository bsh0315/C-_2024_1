####지렁이 게임###

게임 설명
주어진 칸 안에서 정해진 규칙에 따라 먹이를 먹으면서 지렁이를 최대한 키우는 게임입니다.

규칙
=> 벽에 닿거나 지렁이가 자기 자신의 꼬리에 닿으면 안됩니다.
=> 만약 벽에 덯거나 지렁이가 자신의 꼬리에 닿으면 게임이 즉시 종료됩니다.
=> 먹이를 하나 먹을 때마다 플레이어는 포인트 10점을 얻게 됩니다
=> 먹이를 하나 먹을 때마다 지렁이는 더 커지게 됩니다.
=> 지렁이를 움직일 땐 w,a,s,d 키를 이용합니다.
